# AZ Claude Chat

A local Claude chat UI that routes through AstraZeneca's AI Gateway. Requires AZ VPN.

## Setup

1. Clone the repo onto your AZ Mac
2. Run `npm install`
3. Copy `.env.example` to `.env` and add your API key
4. Run `npm start`
5. Open http://localhost:3737

## Requirements
- Node.js 18+
- AZ VPN connected
- Valid AI Gateway subscription key (from portal.ai-gateway.astrazeneca.net)
